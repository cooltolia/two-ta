Описание

Сумка Bukowski сделана нами специально для мужчин, ценящих практичность. Обладая исключительно мужественным дизайном, эта модель сочетает в себе универсальность небольшой дорожной сумки с удобными размерами сумки на каждый день. Кожа с эффектом старения - при использовании остаются естественные потёртости, которые со временем придают изделию благородный вид и индивидуальность! При желании все следы легко можно убрать при помощи обычной губки и бесцветного воска для кожи!

детали

  • Уход
    Протирать сухой мягкой тканью. Обрабатывать универсальным гелем SAPHIR.
